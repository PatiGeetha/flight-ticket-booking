# ✈️ Flight Ticket Booking System

A modern **Flight Ticket Booking Web Application** built using **React, Vite, and Firebase**.
This application allows users to search flights, view available options, and simulate ticket booking through a responsive web interface.

---

## 🚀 Features

* 🔍 Search flights between locations
* 📅 Select travel date
* ✈️ View available flights
* 🧾 Booking interface simulation
* 🔐 Firebase authentication (Google Sign-In)
* ☁️ Firebase hosting deployment
* 📱 Responsive UI for desktop and mobile

---

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/flight-booking-app.git
```

Go to the project directory

```bash
cd flight-booking-app
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```
